## 前言

Node.js 是一个 JavaScript 运行时（runtime），可以使用它运行 JavaScript 代码。它是学习前端开发的必备工具之一。

## 安装 Node.js

1. 访问 [Node.js 官网](https://nodejs.org/) 下载适合 Windows 的安装包。
2. 运行安装程序，按照提示完成安装。

## 验证安装

打开命令提示符（CMD），输入以下命令：

```bash
node -v
```

如果成功安装，将显示 Node.js 的版本号。

## 使用 Node.js

在命令提示符中输入 `node`，进入 Node.js REPL（交互式解释器），可以直接输入 JavaScript 代码进行测试。

```javascript
> console.log("Hello, Node.js!");
Hello, Node.js!
```
