VuePress 是一个基于 Vue.js 的静态网站生成器，Plume 主题是为 VuePress
提供的一种美观且功能丰富的主题。

<Callout type="success">
  接下来我将用 **「VP」** 指代 VuePress，**「羽毛主题」** 指代 Plume 主题。
</Callout>

## 前置条件

- 已安装 Node.js 和 npm
- VuePress 基础了解
- GitHub 账号

## 步骤详解

在本地新建一个 VP 项目

```bash
npm create vuepress-theme-plume@latest
```

在 GitHub 上[创建一个仓库](https://github.com/new)

![](/tutorials/github-create-a-new-repository.png)
