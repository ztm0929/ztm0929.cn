# Next.js 入门教程

Next.js 是一个强大的 React 框架，支持服务端渲染和静态站点生成。

## 安装

首先创建一个新的 Next.js 项目：

```bash
npx create-next-app@latest my-app
cd my-app
npm run dev
```

## 页面路由

Next.js 使用基于文件系统的路由：

- `pages/index.js` → `/`
- `pages/about.js` → `/about`
- `pages/blog/[slug].js` → `/blog/:slug`
