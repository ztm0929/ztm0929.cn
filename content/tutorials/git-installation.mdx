## 前言

Git 是一个分布式版本控制系统，可以帮助开发者管理代码的版本。它是现代软件开发中不可或缺的工具之一。

## 安装 Git

1. 访问 [Git 官网](https://git-scm.com/) 下载适合 Windows 的安装包。
2. 运行安装程序，按照提示完成安装。

## 验证安装

打开命令提示符（CMD），输入以下命令：

```bash
git --version
```

如果成功安装，将显示 Git 的版本号。

## 使用 Git

在命令提示符中输入 `git`，可以查看 Git 的使用帮助和可用命令。

```bash
git init
```

## 结语

恭喜你成功安装并使用了 Git！接下来可以尝试使用 Git 管理你的代码项目。
