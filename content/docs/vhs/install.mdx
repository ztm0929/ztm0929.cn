## 开始之前

- [ ] 拥有良好的网络环境（已开启 系统代理 和 终端代理）
- [ ] 了解软件包管理器（已安装 Windows 的 Scoop 或 macOS 的 Homebrew）

## 动手操作

### 1. 环境预检

### 2. 安装前置依赖

ttyd 和 ffmpeg 是 VHS 的核心组件，确保它们已正确安装：

```zsh title="Terminal.app（终端）"
brew install ttyd ffmpeg
```
### 3. 安装 VHS

```zsh title="Terminal.app（终端）"
brew install vhs
```
