## 我们的目标

<Callout type="success">
在输入几行命令后，**成功在终端返回 Neovim 的版本号**
</Callout>

## 开始之前

- [ ] 拥有良好的网络环境（已开启 [系统代理](/docs/naas/system-proxy) 和 [终端代理](/docs/naas/terminal-proxy)）

- [ ] 了解软件包管理器（已安装 [Windows 的 Scoop](/docs/toolchain/package-managers/scoop/install) 或 [macOS 的 Homebrew](/docs/toolchain/package-managers/homebrew/install)）

### 2. 安装 Neovim

<Tabs groupId="os" items={['Windows', 'macOS']} persist>

<Tab value="Windows">

```powershell title="Windows Terminal（终端）"
scoop install neovim
```

</Tab>

<Tab value="macOS">

```sh title="Terminal.app（终端）"
brew install neovim
```
</Tab>

</Tabs>
