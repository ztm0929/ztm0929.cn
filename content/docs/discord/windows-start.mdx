`Update.exe` 默认不走代理，而我们的网络环境无法直连 Discord 服务器，最终导致启动时卡死在 checking for updates 的界面。

好在有这个开源项目可以帮我们临时解决卡死的问题：

https://github.com/aiqinxuancai/DiscordProxyStart

## 开始之前

- [ ] 良好的网络环境（已运行 [小猫咪](../../naas/system-proxy/clash-verge-rev/)）
