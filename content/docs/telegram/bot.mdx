Telegram 机器人是 Telegram 平台上的自动化程序，可以执行各种任务，如发送消息、处理用户输入、提供信息等。

## 我们的目标

## 开始之前

- [ ] 拥有良好的网络环境（已开启 [系统代理](/docs/naas/system-proxy)）
- [ ] 拥有 Telegram 账号

## 动手操作

### 1. 创建 Telegram 机器人

1. 打开 Telegram 应用，搜索并打开 [BotFather](https://t.me/BotFather)。
2. 点击 "Start" 按钮，开始与 BotFather 交互。
3. 输入 `/newbot` 命令，按照提示为机器人选择一个名字和用户名。
4. 创建成功后，BotFather 会提供一个 API 令牌（Token），请妥善保存这个令牌，因为它是我们与 Telegram 机器人进行通信的关键。
