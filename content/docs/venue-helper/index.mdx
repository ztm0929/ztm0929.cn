[venue-helper](https://github.com/Matt-Dong123/tools4szu/tree/main/venue-helper) 是一个基于 Python 的自动预约工具，能够帮助深大同学自动预约体育场馆。它使用了 requests 来向订票接口发起请求，实现免登录预约。

## 项目结构

```
venue-helper/
├── /data
|   └── RoomList.xlsx
|   └── TimeList.xlsx
|   └── activities.xlsx
|   └── venue.xlsx
├── README.md
├── apis.py
├── download.py
├── main.py
└── settings.py
```
