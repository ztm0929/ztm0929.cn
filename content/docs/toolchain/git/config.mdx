## 动手操作

### 1. 设置 Git 用户信息

Git 需要知道我们的名字和邮箱地址，以便在提交代码时记录这些信息。可以使用以下命令来设置全局的 Git 用户信息：

```bash
git config --global user.name "ztm0929"
git config --global user.email "ztm0929@icloud.com"
```

示例中的 `ztm0929` 和 `ztm0929@icloud.com` 请替换为实际的名字和邮箱地址。
