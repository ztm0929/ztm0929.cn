## 开始之前

- [ ] 良好的网络环境
- [ ] 终端基础操作知识
- [ ] 已安装 Scoop 或 Homebrew

## 动手操作

### 1. 检查系统是否已安装 Git

```
git --version
```

## 安装 Git

1. 访问 [Git 官网](https://git-scm.com/) 下载适合 Windows 的安装包。
2. 运行安装程序，按照提示完成安装。

## 验证安装

打开命令提示符（CMD），输入以下命令：

```bash
git --version
```

如果成功安装，将显示 Git 的版本号。

## 使用 Git

在命令提示符中输入 `git`，可以查看 Git 的使用帮助和可用命令。

```bash
git init
```

## 结语

恭喜你成功安装并使用了 Git！接下来可以尝试使用 Git 管理你的代码项目。
