Windows Terminal 是微软官方推出的终端工具，支持多标签页、分屏、主题等功能，适用于 Windows 10 和 Windows 11。它可以同时运行 PowerShell、命令提示符（CMD）和 Windows Subsystem for Linux（WSL）等多种 Shell。
