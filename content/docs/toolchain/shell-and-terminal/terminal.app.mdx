Terminal.app 是 macOS 系统自带的终端程序，我们可以在其中输入许多命令实现自己的任务需求。


```zsh title="Terminal.app（终端）"
touch ~/.hushlogin
```

`~/.hushlogin` 是一个「标志文件」，创建它将能够实现安静登录，不显示提示信息

```zsh title="Terminal.app（终端）"
Last login: Fri May 29 00:27:09 on ttys000
```
