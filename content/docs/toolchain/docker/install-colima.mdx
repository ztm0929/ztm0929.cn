title: "如何安装 Colima"
---

<DraftNotice />

Colima 是一个在 macOS 上运行 Linux 容器的工具，提供了一个轻量级的虚拟化环境。

## 开始之前

- [ ] 拥有良好的网络环境（已开启 终端代理）
- [ ] 了解软件包管理器（已安装 Homebrew）

## 动手操作

```
brew install docker
```

### 1. 安装 Colima

```zsh
brew install colima
```

### 2. 启动 Colima

```zsh
colima start
```

### 3. 验证安装

```zsh
docker version
```
