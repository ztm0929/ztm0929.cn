GitHub Copilot 是 GitHub 推出的 AI 编程助手，可以在编写代码时提供智能建议，极大地提升开发效率。它有多种形态，我们在这里介绍其中的 VS Code 扩展。

## 我们的目标

<Callout type="success">
安装并启用 GitHub Copilot 插件，**成功通过对话，自动创建一份简易 Python 代码**
