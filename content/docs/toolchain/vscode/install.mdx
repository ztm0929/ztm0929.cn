<DraftNotice />

## 开始之前

- [ ] 拥有良好的网络环境（已开启 [终端代理](../../../naas/terminal-proxy)）
- [ ] 了解软件（包）管理器（已安装 [Windows 的 Scoop](../../package-managers/scoop/install) 或 [macOS 的 Homebrew](../../package-managers/homebrew/install)）

## 动手操作

### 1. 检查网络环境

```powershell title="Windows Terminal（终端）"
Write-Host "【代理环境变量】"; $env:http_proxy; $env:https_proxy
```
应该会返回类似下面的信息：
```
【代理环境变量】
http://127.0.0.1:7897
http://127.0.0.1:7897
```

### 2. 输入命令安装 VS Code

<Tabs groupId="os" items={['Windows', 'macOS']} persist>

<Tab value="Windows">

```powershell title="Windows Terminal（终端）"
scoop install vscode
```

</Tab>

<Tab value="macOS">

```zsh title="Terminal.app（终端）"
brew install --cask visual-studio-code
```

</Tab>

</Tabs>





### 3. 验证安装成功

<Tabs groupId="os" items={['Windows', 'macOS']} persist>

<Tab value="Windows">

```powershell title="Windows Terminal（终端）"
code --version
```
应该会返回类似下面的版本信息：
```
1.114.0
e7fb5e96c0730b9deb70b33781f98e2f35975036
x64
```
</Tab>
<Tab value="macOS">

```zsh title="Terminal.app（终端）"
code --version
```
应该会返回类似下面的版本信息：
```
1.114.0
e7fb5e96c0730b9deb70b33781f98e
2f35975036
x64
```
</Tab>
</Tabs>
