## 开始之前

- [ ] 良好的网络环境
- [ ] 已安装 [Scoop](../../../package-managers/scoop)（Windows）或 [Homebrew](../../../package-managers/homebrew)（macOS）

## 动手操作

```powershell title="Windows Terminal（终端）"
scoop install uv
```

```sh title="Terminal.app（终端）"
brew install uv
```
