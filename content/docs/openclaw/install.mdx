## 开始之前

- [ ] 拥有良好的网络环境（已开启 [终端代理](../../naas/terminal-proxy)）
- [ ] 了解软件（包）管理器（已安装 [Windows 的 Scoop](../../toolchain/package-managers/scoop/install) 或 [macOS 的 Homebrew](../../toolchain/package-managers/homebrew/install)）
- [ ] 了解 JavaScript/TypeScript 生态（已安装 [Node.js](../../frontend/nodejs/install) 和 [pnpm](../../frontend/nodejs/pnpm)）

## 动手操作

### 1. 环境预检

<Tabs groupId="os" items={['Windows', 'macOS']} persist>

<Tab value="Windows">

```powershell title="Windows Terminal（终端）"
$Env:HTTP_PROXY; $Env:HTTPS_PROXY; `
node --version; `
pnpm --version;
```

```
http://127.0.0.1:7897
http://127.0.0.1:7897
v26.1.0
11.0.8
```

</Tab>

<Tab value="macOS">

```zsh title="Terminal.app（终端）"
echo "【代理环境变量】"; echo $HTTP_PROXY; echo $HTTPS_PROXY; `
echo "\n【Node.js 版本】"; node --version; `
echo "\n【pnpm 版本】"; pnpm --version;
```

</Tab>

</Tabs>

### 2. 安装 OpenClaw

<Tabs groupId="os" items={['Windows', 'macOS']} persist>

<Tab value="Windows">

```powershell title="Windows Terminal（终端）"
pnpm add -g openclaw@latest `
	--allow-build=@google/genai `
	--allow-build=koffi `
	--allow-build=openclaw `
	--allow-build=protobufjs `
	--allow-build=tree-sitter-bash
```

</Tab>

<Tab value="macOS">

```zsh title="Terminal.app（终端）"
pnpm add -g openclaw@latest \
	--allow-build=@google/genai \
	--allow-build=koffi \
	--allow-build=openclaw \
	--allow-build=protobufjs \
	--allow-build=tree-sitter-bash
```

</Tab>

</Tabs>

这条安装命令会执行非常多的动作，安装过程中可能会有大量输出，我们耐心等待安装完成，最后会看到类似下面的输出作为结尾：

```
global:
+ openclaw 2026.5.6 (2026.5.7 is available)

Done in 19.5s using pnpm v11.0.8
```

### 3. 验证安装

```zsh
openclaw --version
```

如果成功安装，将显示 OpenClaw 的版本号。

```
OpenClaw 2026.5.6 (c97b9f7)
```
