title: "如何在 Windows 安装 Node.js"
---

## 前置条件

- [Scoop](../../scoop) 已安装并了解基础
- 良好的网络环境（[终端代理](../../naas/terminal-proxy/windows)）

<div className='fd-steps [&_h3]:fd-step'>

### 安装

```powershell title="Windows Terminal（终端）"
scoop install nodejs
```

可以先使用 `scoop search nodejs` 来查看 Scoop 中有哪些 Node.js 版本可用。

### 确认

```powershell title="Windows Terminal（终端）"
node -v
```
