---
title: "如何在 Windows 安装 pnpm"
---

<DraftNotice />

pnpm 是一个依赖管理器，能够帮助我们管理项目所需依赖。

## 前置条件

- [Node.js](../../frontend/node) 已安装并了解基础
- 良好的网络环境（[终端代理](../../naas/terminal-proxy/windows)）
- Scoop 已安装并了解基础

## 步骤

<div className='fd-steps [&_h3]:fd-step'>

### 安装

```powershell title="Windows Terminal（终端）"
scoop install pnpm
```

### 确认

```powershell title="Windows Terminal（终端）"
pnpm -v
```

</div>
