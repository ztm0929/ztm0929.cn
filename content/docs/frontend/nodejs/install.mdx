Node.js 是一个 JavaScript 运行时（runtime），可以使用它运行 JavaScript 代码。它是学习前端开发的必备工具之一。

## 我们的目标

<Callout type="success">
**成功在终端返回 Node.js 版本号**
</Callout>

## 开始之前

- [ ] 已安装 [Scoop](../../../toolchain/package-managers/scoop/install)（Windows） 或 [Homebrew](../../../toolchain/package-managers/homebrew/install)（macOS）

## 动手操作

### 1. 检查

有时我们可能已经安装或系统内置了 Node.js，可以先检查一下：

```
node -v
```

### 2. 安装 Node.js

```
scoop install nodejs
```

```
brew install node
```

### 3. 验证安装

打开命令提示符（CMD），输入以下命令：

```bash
node -v
```

如果成功安装，将显示 Node.js 的版本号。
