title: 如何在本地运行 Fumadocs
---

## 前置条件

- Node.js 安装并了解基础
- pnpm 安装并了解基础
- Markdown 了解基础

```powershell title="Windows Terminal（终端）"
"http_proxy","https_proxy","HTTP_PROXY","HTTPS_PROXY" | % { [System.Environment]::SetEnvironmentVariable($_,"http://127.0.0.1:7897","User") }
```

**开启新的终端窗口**，输入以下命令：

```powershell title="Windows Terminal（终端）"
$env:HTTP_PROXY; $env:HTTPS_PROXY;
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser; Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression;
scoop install git
scoop update
scoop bucket add extras
scoop install nodejs pnpm vscode
```

```powershell title="Windows Terminal（终端）" tab="pnpm"
pnpm create fumadocs-app
```

```powershell title="Windows Terminal（终端）" tab="npm"
npm create fumadocs-app
```
