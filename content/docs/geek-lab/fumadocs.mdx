---
title: 使用 Fumadocs 搭建文档网站
description: Fumadocs 是一款开源文档框架，通过 Next.js 实现。
---

[Fumadocs](https://fumadocs.dev) 是 GitHub 社区里一款比较新颖的文档框架，相比于其他已有的框架（Docusaurus、VitePress
等），它的默认 UI 质感非常好，同时深度集成了 [Next.js](https://nextjs.org/) 的特性，可以尝鲜部署一下。

```shell tab="pnpm"
pnpm create fumadocs-app
```

```shell tab="npm"
npm create fumadocs-app
```
