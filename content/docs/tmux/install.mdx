[tmux](https://github.com/tmux/tmux) 是一个终端复用器，可以让我们在一个终端窗口中同时运行多个会话。

核心概念区分：会话（session）、窗口（window）、面板（pane）

## 开始之前

- [ ] 拥有良好的网络环境（已开启 终端代理）
- [ ] 了解软件包管理器（已安装 Homebrew）

## 动手操作

```zsh title="Terminal.app（终端）"
brew install tmux
```

```zsh title="Terminal.app（终端）"
tmux
```
