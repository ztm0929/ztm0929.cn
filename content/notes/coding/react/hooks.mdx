# React Hooks 入门

学习如何使用 React Hooks 来管理组件状态和副作用。

## useState

`useState` 是最常用的 Hook：

```jsx
import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        增加
      </button>
    </div>
  );
}
```
