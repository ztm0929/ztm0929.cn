# JavaScript 基础语法

这是一个关于 JavaScript 基础语法的教程。

## 变量声明

在 JavaScript 中，你可以使用 `let`、`const` 和 `var` 来声明变量：

```javascript
let name = "张三";
const age = 25;
var city = "北京";
```

## 函数

JavaScript 支持多种函数声明方式：

```javascript
// 函数声明
function greet(name) {
  return `Hello, ${name}!`;
}

// 箭头函数
const greet2 = (name) => `Hello, ${name}!`;
```
