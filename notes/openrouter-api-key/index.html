如何获取 OpenRouter API Key

OpenRouter 是一个模型聚合商，它通常会提供一些模型的免费试用额度，我们可以注册账号并获取 API Key 来使用这些模型。

我们的目标

通过少量的点击与输入操作，成功获取 OpenRouter 的 API Key，它的形式类似于 sk-or-v1-d6ca26d5bf5e3f1330d9ace82dbf66400c5c1d20c53fb368f453e6xxxxxxxxxx

开始之前

  • 拥有良好的网络环境（已运行 小猫咪）

动手操作

注册账号

访问 OpenRouter 官网，点击右上角的 "Sign Up"（注册）按钮，按照提示完成注册流程。

如果已有 GitHub 或 Google 账号，也可以选择使用这些账号直接登录。

创建 API Key

点击 https://openrouter.ai/workspaces/default/keys 进入 API Keys 页面，右上角 New Key（新建密钥）按钮，按照提示创建一个新的 API Key。

填写基本信息

在创建 API Key 的过程中，需要填写一些基本信息：

  • Name（名称）：为 API Key 起一个名字，方便识别
  • Credit limit（额度限制）：我们暂时留空
  • Reset limit every...（重置限制周期）：因为上面的额度限制留空了，这里保持默认的 N/A 即可
  • Expiration（过期时间）：我们暂时保持默认的 No Expiration（永不过期）

保存到本地

用记事本或其他文本编辑器打开一个新的文件，将生成的 API Key 粘贴进去，保存为 openrouter-api-key.txt，并妥善保管这个文件。
