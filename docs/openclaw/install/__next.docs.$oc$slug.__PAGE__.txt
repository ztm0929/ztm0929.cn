title: "如何安装 OpenClaw"
---

<DraftNotice />

## 我们的目标

<Callout type="success">
通过输入少量的命令，**成功在终端显示 OpenClaw 的版本号**，这意味着我们成功安装了 OpenClaw
</Callout>

## 开始之前

- [ ] 拥有良好的网络环境（已开启 [系统代理](../../naas/system-proxy) 和 [终端代理](../../naas/terminal-proxy)）
- [ ] 了解软件（包）管理器（已安装 [Windows 的 Scoop](../../toolchain/package-managers/scoop/install) 或 [macOS/WSL 的 Homebrew](../../toolchain/package-managers/homebrew/install)）
- [ ] 了解 JavaScript/TypeScript 生态（已安装 [Node.js](../../frontend/nodejs/install) 和 [pnpm](../../frontend/nodejs/pnpm)）
- [ ] 如果是 Windows 系统，建议先安装 [WSL](/docs/toolchain/shell-and-terminal/windows-terminal/wsl/install)，以获得更好的兼容性和性能

## 动手操作

### 1. 环境预检

<Tabs groupId="os" items={['Windows', 'macOS', 'WSL']} persist>

<Tab value="Windows">

```powershell title="Windows Terminal（终端）"
$Env:HTTP_PROXY; $Env:HTTPS_PROXY; `
node --version; `
pnpm --version;
```

</Tab>

<Tab value="macOS">

```zsh title="Terminal.app（终端）"
echo $http_proxy; echo $https_proxy;
node --version
pnpm --version
```

</Tab>

<Tab value="WSL">

```bash title="Windows Terminal（终端）"
echo $http_proxy; echo $https_proxy;
node --version
pnpm --version
```

</Tab>

</Tabs>

理想情况下，终端应该会输出以下内容：

```
http://127.0.0.1:7897
http://127.0.0.1:7897
v26.1.0
11.0.8
```

- 第一行和第二行的 `http://127.0.0.1:xxxx` 是我们设置的终端代理地址，具体的端口号因人而异
- `v26.1.0` 是 Node.js 的版本号，越高越好
- `11.0.8` 是 pnpm 的版本号，越高越好

### 2. 安装 OpenClaw

<Tabs groupId="os" items={['Windows', 'macOS', 'WSL']} persist>

<Tab value="Windows">

```powershell title="Windows Terminal（终端）"
pnpm add -g openclaw@latest `
	--allow-build=@google/genai `
	--allow-build=koffi `
	--allow-build=openclaw `
	--allow-build=protobufjs `
	--allow-build=tree-sitter-bash
```

</Tab>

<Tab value="macOS">

```zsh title="Terminal.app（终端）"
pnpm add -g openclaw@latest \
	--allow-build=@google/genai \
	--allow-build=koffi \
	--allow-build=openclaw \
	--allow-build=protobufjs \
	--allow-build=tree-sitter-bash
```

</Tab>

<Tab value="WSL">

```bash title="Windows Terminal（终端）"
pnpm add -g openclaw@latest \
	--allow-build=@google/genai \
	--allow-build=koffi \
	--allow-build=openclaw \
	--allow-build=protobufjs \
	--allow-build=tree-sitter-bash
```

</Tab>

</Tabs>

这条安装命令会执行非常多的动作，安装过程中可能会有大量输出，我们耐心等待安装完成，最后会看到类似下面的输出作为结尾：

```
global:
+ openclaw 2026.5.6 (2026.5.7 is available)

Done in 19.5s using pnpm v11.0.8
```

### 3. 验证安装

```zsh
openclaw --version
```

如果成功安装，将显示 OpenClaw 的版本号。

```
OpenClaw 2026.5.6 (c97b9f7)
```
