### 2. 获取源码并初始化

```powershell title="Windows Terminal（终端）"
git clone https://github.com/Matt-Dong123/tools4szu.git
```

由于项目没有自动依赖文件（`requirements.txt`），我们需要手动安装脚本运行所需的 `requests`、`pandas` 和 `openpyxl` 这三个库。

```powershell title="Windows Terminal（终端）"
cd tools4szu/venue-helper
python -m venv venv
venv\Scripts\activate
pip install requests pandas openpyxl
```

安装完成后，建议运行 `pip list` 检查依赖是否安装成功。你应该能看到如下列表，如果缺少某个库，请重新运行安装命令。

pip list

### 3. 配置购票参数

<Accordions>
<Accordion title="具体列表">

详细字段介绍

- `CGDM` 和 `XMDM` 分别是场馆代码和项目代码，示例中的 `009` 和 `004` 分别就对应着海边网球场和网球项目
- `CDWID` 则是场地 ID，`89d7005a36f341a39d89354d659b639d` 是海边网球场 1 号场，`2b375360778a4978922e48b34d772947` 是海边网球场 2 号场
- `XQWID` 是校区 ID，`1` 是粤海校区，`2` 是丽湖校区
- `KYYSJD` 是可预约时间段，格式为 `HH:00-HH:00`，`YYRQ` 字段是预约日期，需要自行修改示例中的日期！

## 总结回顾

总结一下，我们要做的事情就是：

1. **检查环境**：确保网络代理和必要的软件（Git、Python）已正确安装和配置。
2. **获取源码并安装依赖**：从 GitHub 克隆项目，创建虚拟环境，并安装所需的 Python 库。
3. **配置购票参数**：根据目标场馆和项目，修改脚本中的参数以匹配你想预约的时间和场地。
4. **运行脚本**：执行 Python 脚本，让它自动完成购票流程。

## 常见问题

<Accordions>
<Accordion title="与浏览器自动化相比，脚本的特点是什么？">
由于不需要处理界面，只需要向目标接口发起网络请求，因此速度更快，效率更高。但行为特征较明显，服务器更容易识别为自动化操作。
</Accordion>
<Accordion title="为什么预约失败？">
Cookie 的有效期通常只有 3~4 小时。建议在抢票前半小时内重新登录并获取最新 Cookie，以确保脚本能正常工作。
</Accordion>
</Accordions>
