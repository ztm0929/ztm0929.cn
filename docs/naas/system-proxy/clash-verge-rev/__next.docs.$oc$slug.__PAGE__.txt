Clash Verge Rev 是一个网络代理工具（俗称「小猫咪」），我们可以利用它实现网络优化。

## 我们的目标

<Callout type="success">
通过启用小猫咪，**成功在浏览器访问 GitHub、Hugging Face 和 YouTube 等网站**
</Callout>

## 动手操作

### 1. 环境预检

先尝试直接使用浏览器访问 [GitHub](https://github.com)、[Hugging Face](https://huggingface.co) 和 [YouTube](https://www.youtube.com)，如果无法访问则继续下一步。

<Callout type="warning">
预期情况下，HF 和 YouTube 通常是无法访问的，而 GitHub 有时可以访问，但我们依然需要优化网络。
</Callout>

- GitHub：https://github.com
- Hugging Face：https://huggingface.co
- YouTube：https://www.youtube.com

GitHub 几乎是编程学习的必经之地，其中有大量优秀开发者的代码可供学习；Hugging Face 是 AI 领域的资源宝库，而 YouTube 则有许多编程教学视频。

### 1. 下载并安装

点击下面的链接下载「小猫咪」在对应平台的安装包：

- [Windows 安装包](https://assets.ztm0929.host/docs/naas/Clash.Verge_2.4.7_x64-setup.exe)
- [macOS 安装包](https://assets.ztm0929.host/docs/naas/Clash.Verge_2.4.7_aarch64.dmg)

如果你了解 PowerShell，可以直接运行以下命令快速静默安装：
```powershell
.exe /S /R
```

如果你还是更喜欢图形界面安装，也可以双击运行下载的安装包，按照提示完成安装：
- 可能会弹出「用户帐户控制」的提示，点击「是」继续安装
- 安装语言选择中文
- 安装路径保持默认即可，点击「下一步」继续
- 完成完成后运行

### 2. 导入订阅

- 在「设置」页面启用「DNS 覆写」并关闭「IPv6」
- 在「订阅」页面输入链接并点击「导入」，订阅链接输入私密信息，私信天明获取
- 导入成功后在「代理」页面应该能看到许多节点

### 3. 开启基础选项

- 系统代理
- 开机自启
- 静默启动

### 4. enjoy!

- 此时应该能够流畅访问 GitHub、HugingFace 等网站了
