Clash Verge Rev 是一个网络代理工具，我们可以利用它实现网络优化。

## 步骤说明

<div className='fd-steps [&_h3]:fd-step'>

### 下载

访问 [NaaS 项目说明页面](https://ztm0929.net/en/docs/network-utilities/clash-verge-rev/)，下载 Clash Verge Rev 的最新版本。

### 安装

如果你了解 PowerShell，可以直接运行以下命令快速静默安装：
```powershell
.exe /S /R
```

如果你还是更喜欢图形界面安装，也可以双击运行下载的安装包，按照提示完成安装：
- 可能会弹出「用户帐户控制」的提示，点击「是」继续安装
- 安装语言选择中文
- 安装路径保持默认即可，点击「下一步」继续
- 完成完成后运行

### 导入订阅

- 在「设置」页面启用「DNS 覆写」并关闭「IPv6」
- 在「订阅」页面输入链接并点击「导入」，订阅链接输入私密信息，私信天明获取
- 导入成功后在「代理」页面应该能看到许多节点

### 开启基础选项

- 系统代理
- 开机自启
- 静默启动

### enjoy!

- 此时应该能够流畅访问 GitHub、HugingFace 等网站了

</div>
